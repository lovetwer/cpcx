# UniPush 推送功能使用说明

## 功能概述

本项目已集成 UniPush 推送功能，支持用户自定义推送时间，实现个性化的买票提醒和开奖通知。

## 页面结构更新

### 新增设置页面
- **路径**: `pages/settings/settings.vue`
- **功能**: 集中管理推送设置和账户设置
- **入口**: 从"我的"页面点击"设置"按钮进入

### "我的"页面调整
- **变更**: 将"退出登录"按钮改为"设置"按钮
- **目的**: 简化页面结构，将设置功能集中管理

## 推送功能特性

### 1. 买票提醒
- **功能**: 在指定时间提醒用户选号购买彩票
- **默认时间**: 
  - 下午 17:00
  - 晚上 21:00
- **自定义**: 用户可以添加、修改、删除提醒时间（最多4个）
- **触发条件**: 当日未选号时发送提醒

### 2. 开奖提醒
- **功能**: 在开奖后推送中奖结果
- **默认时间**:
  - 晚上 22:00
  - 晚上 23:00
- **自定义**: 用户可以添加、修改、删除提醒时间（最多4个）
- **推送内容**:
  - 有中奖：推送具体中奖信息
  - 无中奖：推送"再接再厉"鼓励信息

## 使用方法

### 访问设置页面
1. 打开应用，进入"我的"页面
2. 点击右上角"设置"按钮
3. 进入设置页面

### 配置推送设置
1. **启用推送**: 打开"启用推送"开关
2. **买票提醒设置**:
   - 打开"买票提醒"开关
   - 点击时间旁边的"修改"按钮调整时间
   - 点击"+ 添加提醒时间"增加新的提醒时间
   - 点击"删除"按钮移除不需要的时间
3. **开奖提醒设置**:
   - 打开"开奖提醒"开关
   - 同样可以修改、添加、删除提醒时间

### 测试推送功能
1. 在设置页面找到"推送测试"选项
2. 点击"发送测试"按钮
3. 检查是否收到测试推送消息

### 退出登录
1. 在设置页面底部找到"账户管理"部分
2. 点击"退出"按钮
3. 确认退出登录

## 技术实现

### 文件结构
```
pages/
├── settings/
│   └── settings.vue          # 设置页面
├── mine/
│   └── mine.vue              # 我的页面（已修改）
└── index/
    └── index.vue             # 首页（移除推送设置）

utils/
├── push-service.js           # 推送服务（支持自定义时间）
├── push-config.js            # 推送配置
└── push-test.js              # 推送测试工具

components/
└── push-settings/            # 推送设置组件（保留备用）
    └── push-settings.vue
```

### 推送时间配置
推送服务支持动态配置推送时间：

```javascript
// 示例配置
const settings = {
  pushEnabled: true,
  buyReminderEnabled: true,
  buyReminderTimes: [
    { hour: 17, minute: 0 },  // 下午5点
    { hour: 21, minute: 0 }   // 晚上9点
  ],
  lotteryReminderEnabled: true,
  lotteryReminderTimes: [
    { hour: 22, minute: 0 },  // 晚上10点
    { hour: 23, minute: 0 }   // 晚上11点
  ]
}
```

### 数据存储
- 推送设置保存在本地存储中：`uni.getStorageSync('pushSettings')`
- 设置更改后自动同步到推送服务

## 推送消息类型

### 买票提醒消息
- **标题**: "🎫 买票提醒"
- **内容**: "今日还未选号，快来选择您的幸运号码吧！\n点击进入选号页面"

### 中奖通知消息
- **标题**: "🎉 恭喜中奖！"
- **内容**: 显示具体中奖注数和类型

### 鼓励消息
- **标题**: "💪 再接再厉"
- **内容**: "今日暂未中奖，继续努力！明天的幸运就在前方等着您！"

## 注意事项

1. **权限要求**: 需要在 manifest.json 中配置推送权限
2. **平台支持**: 仅在 APP-PLUS 平台支持推送功能
3. **网络要求**: 推送功能需要网络连接
4. **时间精度**: 推送检查间隔为1分钟
5. **重复推送**: 同一时间点每天只推送一次

## 故障排除

### 推送不工作
1. 检查推送开关是否开启
2. 确认应用有推送权限
3. 使用"推送测试"功能验证
4. 检查网络连接状态

### 时间设置无效
1. 确认时间格式正确
2. 检查是否保存设置
3. 重启应用重新初始化推送服务

### 收不到推送
1. 检查系统通知权限
2. 确认应用在后台运行
3. 查看推送服务初始化状态
